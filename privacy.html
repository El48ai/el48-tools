Kebijakan Privasi untuk EL48 Tools

Terakhir diperbarui: 25 November 2025

A. Pendahuluan

Aplikasi EL48 Tools menyediakan berbagai utilitas seperti kompres foto dan peningkatan kualitas gambar. Kebijakan ini menjelaskan bagaimana aplikasi ini menangani data pengguna.

B. Informasi yang Dikumpulkan

  • Data teknis: perangkat, versi OS, dan data error untuk perbaikan bug.
  • Data file/media: aplikasi hanya membaca file gambar yang Anda pilih. Seluruh proses dilakukan secara lokal di perangkat tanpa pengiriman ke server.

Aplikasi tidak mengumpulkan informasi pribadi seperti nama, nomor telepon, atau akun media sosial.

C. Izin Aplikasi

  • Storage / Files and Media: diperlukan agar pengguna dapat memilih gambar yang akan diproses.
  • Kamera: hanya jika fitur ambil foto digunakan.

Izin dapat dicabut kapan saja melalui pengaturan perangkat.

D. Penggunaan Informasi

  • Menjalankan fungsi aplikasi.
  • Memperbaiki bug dan meningkatkan performa.

E. Penyimpanan & Keamanan

Semua pengolahan gambar dilakukan offline di perangkat. Tidak ada file yang dikirim, disimpan, atau dibagikan ke server manapun.

F. Anak-anak

Aplikasi tidak ditujukan untuk anak-anak di bawah usia 13 tahun.

G. Perubahan Kebijakan

Kami dapat memperbarui kebijakan ini sewaktu-waktu. Perubahan akan ditampilkan pada halaman ini.
