Kebijakan Privasi

Terakhir diperbarui: Januari 2026

A. Pendahuluan

EL48 Tools menyediakan utilitas seperti kompres foto dan peningkatan kualitas gambar. Kebijakan ini menjelaskan bagaimana aplikasi ini menangani data pengguna.

🔒 Singkatnya: Kami tidak mengumpulkan, menyimpan, atau mengirim data apapun. Semua proses terjadi 100% di perangkat Anda.

B. Informasi yang Dikumpulkan

EL48 Tools tidak mengumpulkan informasi apapun dari pengguna, termasuk:

  • Data pribadi (nama, email, nomor telepon)
  • File atau foto yang Anda proses
  • Data perangkat atau lokasi
  • Riwayat penggunaan atau analytics

Aplikasi ini berjalan sepenuhnya offline di browser Anda tanpa koneksi ke server manapun.

C. Izin Aplikasi

Satu-satunya akses yang diperlukan:

  • Akses file/foto: hanya saat Anda secara aktif memilih gambar untuk diproses. File tidak pernah dikirim keluar dari perangkat Anda.

D. Penyimpanan & Keamanan

Semua pengolahan gambar dilakukan sepenuhnya di browser menggunakan teknologi Canvas API. Tidak ada file yang dikirim, disimpan di server, atau dibagikan ke pihak manapun.

E. Layanan Pihak Ketiga

Aplikasi ini di-hosting di GitHub Pages. Kunjungi kebijakan privasi GitHub untuk informasi lebih lanjut tentang data yang mungkin dikumpulkan oleh platform hosting.

F. Anak-anak

Aplikasi tidak ditujukan untuk anak-anak di bawah usia 13 tahun.

G. Perubahan Kebijakan

Kami dapat memperbarui kebijakan ini sewaktu-waktu. Perubahan akan ditampilkan pada halaman ini dengan tanggal yang diperbarui.
